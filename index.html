Теоретичне питання

Поясніть своїми словами, як ви розумієте, що таке деструктуризація і навіщо вона потрібна?
Деструктуризація - це механізм за допомогою якого можна розгортати дані. Тобто при розгортанні обєкта в змінну, яка буде відповідати назві ключа запишиться значення ключа, а при розгортанні масиву в змінні запишуться значення масиву в тому порядку в якому вони зазначені в масиві. При деструктуризації зручно також використовувати значення по дефолту, у випадку якщо ці дані будуть відсутні. Також Деструктуризація допомагає переприсвоїти значення не використовуючи додаткових змінних.
Деструктуризація робить роботу більш простою і зменшує кількість помилок, які можна зробити при написанні коду, робить код більш читабельним. 
Приклад використання деструктуризації обєктів:
        const user1 = {
          name: "John",
          years: 30
        };
        const {name, years=33} = user1;
      
Приклад використання деструктуризації масивів:
        const array = ['value', 'showValue'];
        const [value, showValue] = array;
      
Приклад переприсвоєння значення:
        let value = "value";
        let showValue = "showValue";
       [value, showValue] = [showValue, value];
